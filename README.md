# Retail Supply Chain: Inventory Optimization

## Project Overview
In this project, I’m tackling the "Stock-Out" problem. My goal is to identify which products are at risk of running out and analyze supplier performance to ensure the store never has an empty shelf.

## Technical Skills Used
*   **Operational Filtering:** Identifying items below reorder thresholds.
*   **Risk Assessment:** Sorting by urgency to prioritize inventory replenishment.
